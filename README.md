## 🚀 Overview

A **premium personal portfolio** built from scratch with advanced animations, glassmorphism design, and a fully functional Node.js backend for contact form email delivery. Designed to showcase skills, projects, and professional experience in a visually stunning, performance-optimized interface.

> 🎓 Final-year IT Engineering Student — **CGPA: 9.3** | Finolex Academy of Management and Technology

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 **Cinematic Loader** | GSAP-powered clip-path reveal animation on page load |
| 🖱️ **Magnetic Cursor** | Radial glow follows the cursor with GSAP physics |
| ⚡ **Smooth Scroll** | Lenis smooth scroll synced with GSAP ScrollTrigger |
| 🃏 **3D Scroll Reveals** | Cards and sections animate with rotationX/Y on scroll |
| ⌨️ **Typed.js Hero** | Cycling role titles with realistic typing & backspace effect |
| 🔮 **Glassmorphism UI** | Frosted-glass navbar, project cards, and testimonial tiles |
| 📬 **Contact API** | Node.js + Nodemailer backend — sends email on form submit |
| 📱 **Fully Responsive** | Mobile-first layout, custom breakpoints for every section |
| 🎨 **Tech Carousel** | Auto-scrolling Owl Carousel of tech stack icons |
| 💬 **Testimonials** | Real client/mentor reviews with carousel display |

## ⚙️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Sujal-85/Sujal_Portfolio.git
cd Sujal_Portfolio
```

### 2. Install backend dependencies
```bash
cd backend
npm install
```

### 3. Configure environment variables
Create a `.env` file inside `backend/`:
```env
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASS=your_app_password
PORT=5000
```
> ⚠️ Use a [Gmail App Password](https://myaccount.google.com/apppasswords), not your actual Gmail password.

### 4. Run the server
```bash
node server.js
```
The portfolio will be live at `http://localhost:5000`
